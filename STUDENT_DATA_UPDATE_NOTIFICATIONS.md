# نظام إشعارات تحديث بيانات الطالب - Kids Bus App

## 🎯 الهدف
إرسال إشعارات فورية لولي الأمر والمشرف عند تحديث بيانات الطالب من قبل الإدمن.

## ✨ المزايا الجديدة

### 📱 إشعارات ولي الأمر
- ✅ إشعار فوري عند تغيير أي بيانات للطالب
- ✅ تفاصيل التغييرات (من القيمة القديمة إلى الجديدة)
- ✅ اسم الإدمن الذي قام بالتحديث
- ✅ صوت واهتزاز مع الإشعار

### 👨‍🏫 إشعارات المشرف
- ✅ إشعار للمشرف المسؤول عن الباص
- ✅ تنبيه عند تغيير بيانات طالب في باصه
- ✅ تفاصيل التحديثات المهمة
- ✅ صوت واهتزاز مع الإشعار

## 🔧 البيانات المتتبعة

### البيانات الأساسية
- **اسم الطالب** - تغيير الاسم
- **اسم المدرسة** - تغيير المدرسة
- **الصف** - انتقال لصف آخر
- **العنوان** - تغيير عنوان السكن

### بيانات ولي الأمر
- **اسم ولي الأمر** - تحديث الاسم
- **رقم هاتف ولي الأمر** - تغيير رقم الهاتف

### بيانات الباص
- **الباص المخصص** - تسكين أو نقل لباص آخر
- **الحالة الحالية** - تغيير حالة الطالب

### بيانات إضافية
- **ملاحظات** - إضافة أو تعديل ملاحظات

## 🚀 كيفية العمل

### 1. عند تعديل البيانات من الإدمن
```dart
// في EditStudentScreen
await _notificationService.notifyStudentDataUpdate(
  studentId: updatedStudent.id,
  studentName: updatedStudent.name,
  parentId: updatedStudent.parentId,
  busId: updatedStudent.busId,
  updatedFields: updatedFields,
  adminName: adminName,
);
```

### 2. تتبع التغييرات
```dart
final updatedFields = <String, dynamic>{};

if (_student!.name != _nameController.text.trim()) {
  updatedFields['name'] = {
    'old': _student!.name,
    'new': _nameController.text.trim(),
  };
}
```

### 3. إرسال الإشعارات
- **لولي الأمر**: إشعار مباشر بمعرف ولي الأمر
- **للمشرف**: البحث عن المشرف المسؤول عن الباص

## 📋 أمثلة على الإشعارات

### إشعار ولي الأمر
```
العنوان: تحديث بيانات الطالب
المحتوى: تم تحديث بيانات الطالب أحمد محمد من قبل الإدارة

التحديثات:
• اسم الطالب: من "أحمد محمد" إلى "أحمد علي"
• الصف: من "الصف الثاني" إلى "الصف الثالث"
• الباص المخصص: من "غير محدد" إلى "باص رقم 123"
```

### إشعار المشرف
```
العنوان: تحديث بيانات طالب في الباص
المحتوى: تم تحديث بيانات الطالب فاطمة أحمد في الباص الخاص بك

التحديثات:
• اسم المدرسة: من "مدرسة النور" إلى "مدرسة الأمل"
• العنوان: من "شارع الملك فهد" إلى "شارع الأمير محمد"
```

## 🧪 كيفية الاختبار

### 1. اختبار من شاشة الاختبار
1. انتقل إلى: `/test/notification-system`
2. اضغط على "اختبار تحديث بيانات الطالب"
3. تحقق من وصول الإشعارات

### 2. اختبار من الإدمن
1. سجل دخول كإدمن
2. انتقل إلى إدارة الطلاب
3. اختر طالب وقم بتعديل بياناته
4. احفظ التعديلات
5. تحقق من وصول الإشعارات لولي الأمر والمشرف

### 3. اختبار تسكين الباص
1. من شاشة جميع الطلاب
2. اختر "تسكين باص" لطالب
3. اختر باص جديد أو ألغ التسكين
4. تحقق من الإشعارات
