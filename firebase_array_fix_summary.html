إصلاح مشكلة Firebase Arrays

حل مشكلة "FieldValue.serverTimestamp() is not currently supported inside arrays"

وصف المشكلة

الخطأ:

Exception: فشل في مزامنة بيانات الطالب مع ولي الأمر: [cloud_firestore/unknown] Invalid data. FieldValue.serverTimestamp() is not currently supported inside arrays
السبب:

Firebase Firestore لا يدعم استخدام FieldValue.serverTimestamp() داخل Arrays. عندما نحاول إضافة timestamp داخل array في document، يحدث هذا الخطأ.

الحل المطبق

تم تطبيق حلين متكاملين:

  1. استبدال serverTimestamp بـ DateTime: استخدام DateTime.now().toIso8601String() بدلاً من FieldValue.serverTimestamp() داخل Arrays
  2. نظام علاقات منفصل: إنشاء collection منفصل للعلاقات بين الأولياء والطلاب

1 مقارنة الكود قبل وبعد الإصلاح

❌ قبل الإصلاح (يسبب خطأ)
// خطأ: استخدام serverTimestamp داخل array currentChildren.add({ ...childData, 'addedAt': FieldValue.serverTimestamp(), // ❌ خطأ }); await _firestore.collection('users').doc(parentId).update({ 'children': currentChildren, // ❌ يحتوي على serverTimestamp 'updatedAt': FieldValue.serverTimestamp(), });
✅ بعد الإصلاح (يعمل بنجاح)
// ✅ صحيح: استخدام DateTime string currentChildren.add({ ...childData, 'addedAt': DateTime.now().toIso8601String(), // ✅ يعمل }); await _firestore.collection('users').doc(parentId).update({ 'children': currentChildren, // ✅ يحتوي على string timestamps 'updatedAt': FieldValue.serverTimestamp(), // ✅ خارج array });

2 الحل البديل: نظام العلاقات المنفصل

بدلاً من حفظ الأطفال في array داخل document الوالد، تم إنشاء collection منفصل:

// إنشاء علاقة منفصلة في collection جديد final relationshipId = '${parentId}_${student.id}'; await _firestore.collection('parent_children').doc(relationshipId).set({ 'parentId': parentId, 'studentId': student.id, 'studentName': student.name, 'studentGrade': student.grade, 'studentSchool': student.schoolName, 'studentRoute': student.busRoute, 'studentPhoto': student.photoUrl, 'studentQR': student.qrCode, 'studentStatus': student.currentStatus.toString().split('.').last, 'isActive': student.isActive, 'createdAt': FieldValue.serverTimestamp(), // ✅ يعمل خارج array 'updatedAt': FieldValue.serverTimestamp(), });
