إعداد Firebase Storage

دليل شامل لحل مشكلة رفع الصور في تطبيق MyBus

حل سريع للمشكلة

إذا كنت تواجه مشكلة في رفع الصور، اتبع هذه الخطوات السريعة:

  1. اذهب إلى Firebase Console
  2. اختر مشروع mybus-5a992
  3. اذهب إلى Storage → Rules
  4. استبدل القواعد بالقواعد المؤقتة أدناه

1 تطبيق قواعد مؤقتة للاختبار

انسخ هذه القواعد وطبقها في Firebase Console:

rules_version = '2'; service firebase.storage { match /b/{bucket}/o { match /{allPaths=**} { allow read, write: if true; } } }
تحذير: هذه القواعد غير آمنة وللاختبار فقط. لا تستخدمها في الإنتاج!

2 اختبار الاتصال

بعد تطبيق القواعد، اختبر الاتصال:

من التطبيق:
  • شغل التطبيق
  • جرب إضافة طالب مع صورة
  • راقب الـ console للأخطاء
اختبار منفصل:

3 تطبيق قواعد آمنة (بعد نجاح الاختبار)

بعد التأكد من عمل الرفع، طبق هذه القواعد الآمنة:

rules_version = '2'; service firebase.storage { match /b/{bucket}/o { // Student Photos - Allow authenticated users match /student_photos/{imageId} { allow read: if request.auth != null; allow write: if request.auth != null && request.resource.size < 5 * 1024 * 1024 // Max 5MB && request.resource.contentType.matches('image/.*'); } // Bus Photos - Allow authenticated users match /bus_photos/{imageId} { allow read: if request.auth != null; allow write: if request.auth != null && request.resource.size < 10 * 1024 * 1024 // Max 10MB && request.resource.contentType.matches('image/.*'); } // Profile Photos - Allow authenticated users match /profile_photos/{imageId} { allow read: if request.auth != null; allow write: if request.auth != null && request.resource.size < 3 * 1024 * 1024 // Max 3MB && request.resource.contentType.matches('image/.*'); } // Test Images - For testing only match /test_images/{imageId} { allow read, write: if request.auth != null; } // Deny all other access match /{allPaths=**} { allow read, write: if false; } } }

استكشاف الأخطاء

permission-denied

المستخدم غير مصرح له بالرفع

  • ✓ تحقق من تسجيل الدخول
  • ✓ طبق القواعد المؤقتة
  • ✓ تحقق من Authentication
network error

مشكلة في الاتصال بالإنترنت

  • ✓ تحقق من الإنترنت
  • ✓ جرب شبكة مختلفة
  • ✓ تحقق من Firewall
object-not-found

المجلد أو الملف غير موجود

  • ✓ تحقق من مسار الملف
  • ✓ تحقق من صلاحيات القراءة
  • ✓ أنشئ المجلد يدوي<|im_start|>
retry-limit-exceeded

تم تجاوز حد المحاولات

  • ✓ انتظر قليلاً
  • ✓ قلل حجم الصورة
  • ✓ أعد تشغيل التطبيق
